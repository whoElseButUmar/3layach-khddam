# Project Instructions

- Build this project with Cloudflare as the intended hosting/runtime platform.
- Assume the account is on Cloudflare's free tier unless the user says otherwise.
- Prefer Cloudflare Pages, Workers, D1, KV, R2, and related free-tier-compatible services when they fit the task.
- This app is a no-login shared sticky board. Treat the six-letter board code as the primary access identifier.
- Keep the UI modern skeuomorphic: wood board, tactile sticky notes, subtle shadows, and polished micro-interactions.
